FAQ for users
  • Q: Where can I find instructions for using this system?
    A: The slides presented by Tim in Aveiro (April 2012) are here. The main Handlebar user manual (also linked from the main menu above) is here.
  • Q: What should I log into this system?
    A: If you have exposed any organism to any chemical as part of the NanoFATE project then it needs to go in here.
  • Q: What if I don't actually store any samples?
    A: You still need to enter the samples on the system as if you had stored them, because this database needs a record of all exposures. In the near future, headline results like LD50 values will be linked to you records in this database.
  • Q: Do I really need a separate code for each organism?
    A: Only if you are keeping them as individual samples. If you have, say, 20 daphnea in one a tube collected all at once from one exposure then that is just one sample and therefore barcode. The guiding principle is one code per Eppendorf (or whatever you store samples in).
  • Q: What if I kept samples of the exposure medium?
    A: You can record these, but the Experiment Creator form does not (yet) support auto-defining them. You need to allocate the appropriate number of "exposure medium sample" codes from the main interface, then download the spreadsheet and fill it in with the appropriate values.
  • Q: What if I don't have a uniform number of samples for each dose or timepoint? How do I stop the system assigning the extra codes for samples that I never took?
    A: Fill in all your doses, timepoints and tissues plus the maximum number of reps, then paste the unused codes from the spreadsheet into the "Mark codes as being disposed" section on the main web interface, with the comment "samples never created".
  • Q: How do I record details like confounding factors or other experimental details?
    A: You can add these as comments, but bear in mind that this database is never going to be able to hold a comprehensive description of your experimental set-up; it is just an overview recording the most important factors that the exposure studies have in common.
  • Q: What if I used a mixture of nanoparticles in an exposure?
    A: You would need to regard this mixture as being a NP batch in itself, and add your particular mixture-batch to the NP batch table.
  • Q: How do I add or correct things in the drop-down lists?
    A: Use the links on the form, but be sure that you are not changing something you shouldn't. For example, only Kerstin should ever touch the table of nanoparticle samples.
